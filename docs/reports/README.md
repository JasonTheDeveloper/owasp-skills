# Vulnerability Assessment Reports

This directory contains OWASP vulnerability assessment reports generated by
the **vulnerability-scanner** agent (`.github/agents/vulnerability-scanner.md`).

## How to generate a report

Open GitHub Copilot Chat in VS Code and invoke the agent:

```
@vulnerability-scanner Scan this codebase for vulnerabilities and save the report.
```

The agent will:

1. Discover the technology profile of the repository.
2. Select the relevant OWASP skills from `.github/skills/`.
3. Assess the codebase against each applicable OWASP Top 10 framework.
4. Collate all findings.
5. Save a structured Markdown report to this directory as
   `vulnerability-report-YYYY-MM-DD.md`.

## Report naming convention

```
vulnerability-report-YYYY-MM-DD.md
```

Where `YYYY-MM-DD` is the ISO 8601 date on which the report was generated.
